## 🚀 Featured Project

### 🦂 Scorpion — DevSecOps Security Orchestration Platform

> A full-stack DevSecOps automation platform built to orchestrate secure CI/CD workflows, integrating multiple security scanning engines with AI-powered remediation suggestions.

| Feature | Details |
|---|---|
| 🔍 SAST Scanning | Integrated **Semgrep** for automated static code analysis |
| 🐳 Container Scanning | **Trivy** for container vulnerability detection |
| 🔐 Auth | **Supabase Auth** with GitHub & Google OAuth |
| 🤖 AI Remediation | **OpenAI API** for automated vulnerability fix suggestions |
| ⏰ Scheduled Scans | **Node-Cron** + child process execution |
| 🌐 Backend | REST APIs with **Node.js** + **Express.js** |

**Tech Stack:** `React` `TypeScript` `Node.js` `Express.js` `Semgrep` `Trivy` `Supabase` `OpenAI API`
